# nature-skills Hermes Tap

从 [Yuan1z0825/nature-skills](https://github.com/Yuan1z0825/nature-skills) (26.4k ⭐) 适配的 Hermes Agent 技能包，面向科研论文写作、审稿回复、数据分析和图表制作。

## 安装

```bash
hermes skills tap add Delagong950/nature-skills
hermes skills install nature-response  # 安装单个
# 或直接装全部
hermes skills install nature-writing
hermes skills install nature-polishing
hermes skills install nature-figure
hermes skills install nature-reviewer
hermes skills install nature-citation
hermes skills install nature-reader
hermes skills install nature-academic-search
hermes skills install nature-data
hermes skills install nature-downloader
hermes skills install nature-experiment-log
hermes skills install nature-literature-pipeline
hermes skills install nature-paper2ppt
hermes skills install nature-paper-to-patent
hermes skills install nature-proposal-writer
hermes skills install nature-ref-verifier
```

## 技能列表

| Skill | 功能 |
|-------|------|
| nature-response | 审稿意见回复 / rebuttal letter |
| nature-writing | Nature 风格论文写作 |
| nature-polishing | 英文润色 / 中译英 |
| nature-figure | 科研绘图 (Python/R) |
| nature-reviewer | 模拟审稿 (3 位审稿人) |
| nature-citation | CNS 系列引用管理与添加 |
| nature-reader | 论文精读 / 中英对照翻译 |
| nature-academic-search | 多源文献检索 |
| nature-data | 数据统计与分析 |
| nature-downloader | 论文下载 (PMC/arXiv/CNKI) |
| nature-experiment-log | 实验记录 / Obsidian 笔记本 |
| nature-literature-pipeline | 文献流水线 |
| nature-paper2ppt | 论文转演示文稿 |
| nature-paper-to-patent | 论文转专利 |
| nature-proposal-writer | 基金申请书 (NSF/NIH/国自然) |
| nature-ref-verifier | 引用文献验证查错 |
